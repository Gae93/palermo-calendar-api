# Palermo FC Calendar API

API automatica per estrarre il calendario delle partite del Palermo FC.

## Endpoints

- `GET /` - Info API
- `GET /api/partite` - Tutte le partite
- `GET /api/partite/casa` - Solo partite in casa
- `GET /health` - Health check

## Deploy

Deployato automaticamente su Render.com
